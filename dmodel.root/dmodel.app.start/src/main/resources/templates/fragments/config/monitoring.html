Monitoring configuration
Our approach distinguishes between fine granular and coarse granular monitoring. The automated validation, which is woven into our approach, allows switching between fine and coarse granular monitoring at runtime. This works on the service level and if metrics reach certain values, the coarse granular monitoring is disabled for specific services. These thresholds can be configured here.
