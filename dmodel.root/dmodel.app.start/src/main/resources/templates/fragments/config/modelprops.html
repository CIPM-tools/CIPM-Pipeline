Automated Mapping Initializers:
Mapping extraction
Using this utility allows you to extract the mapping between source code elements and architecture model elements based on annotations and comments in the code. In the first step of the procedure, the existing mappings are removed, so please be careful if there are existing mappings.

The configuration must be error-free in order for this procedure to be successful.
Initialize instrumentation model
The following procedure uses the repository model and creates corresponding instrumentation points in the instrumentation model. Be careful using this tool as it overrides the currently used instrumentation model.

The configuration must be error-free in order for this procedure to be successful.
